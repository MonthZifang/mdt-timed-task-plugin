# timed-task-plugin

用于执行定时服务器任务，配置文件位于 `config/mods/config/timed-task-plugin/config.json`。

## 功能

- 定时执行服务器任务
- 适合自动开服、自动广播、定时维护等场景
- 通过配置文件集中管理计划任务

## 配置位置

```text
config/mods/config/timed-task-plugin/config.json
```

## 适用场景

- 开服后自动执行初始化命令
- 定时发送公告
- 定时执行维护类命令
